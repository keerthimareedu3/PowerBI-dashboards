# 📊 Sales Performance Dashboard - Power BI

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze sales performance, profit trends, and regional business metrics. The dashboard helps businesses make data-driven decisions by providing visual insights into key performance indicators.

## 🎯 Objectives
- Analyze sales and profit performance.
- Identify top-performing regions and products.
- Create interactive visualizations for business decision-making.
- Monitor key performance indicators (KPIs).

## 🛠️ Technologies Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Dashboard Features
- Sales Analysis
- Profit Analysis
- Regional Performance
- KPI Cards
- Interactive Filters and Slicers
- Trend Analysis
